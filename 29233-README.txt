===========================================

The files in the \maps and \flags folders
are named by their country's two-character
FIPS-10 code.  

For more information, or if a country 
appears to be missing, refer to the 
Factbook's Appendix D - Cross Reference 
List of Country Data Codes.

There are two lists below, the first sorted
by country name, the second sorted by
country code.
